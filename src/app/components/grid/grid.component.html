Grid

The Grid is a fluid system that shrinks with the browser/device. Containers have a max width of of 72.0rem (1200px), but there are 3 other responsive breakpoints that you can use. Cubed CSS uses the the Flexible Box Layout standard.

Columns

Simply add columns you want in a row, and they'll evenly take up the available space. If you want three columns, then just add three columns. There is no limit on the number of columns, but Cubed CSS only supports up to 12 with responsive breakpoints.

  1. Add a .row container
  2. Add any number of .col elements and they'll all be equal width automatically

Breakpoints

Cubed CSS uses 5 breakpoint variables throughout the framework

Breakpoint Width Device
Extra Large < 1200px or 75rem Large Desktop
Large < 992px or 62rem Desktop
Medium < 768px or 48rem Tablet or Large Phone
Small < 576px or 36rem Phone
